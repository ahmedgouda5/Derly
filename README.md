# Derly

Derly is a dashboard for managing a supermarket or market store. It allows you to track your sales, profits, and generate PDF invoices easily.

---

## Features

- **Sales & Profit Tracking**: Monitor your sales and profits daily and monthly.
- **PDF Invoice Generation**: Create invoices for any sale and save or print them as PDF.
- **Charts for Profits**: Visualize monthly profits using charts.
- **Quarterly View**: Each chart displays profits for four months to divide the year into quarters.
- **User-Friendly Dashboard**: Clean and organized interface for easy store management.

---

## Current Version

- This is **Version 1.0** of the project.
- All core features are working properly.
- **Note:** No backend is implemented yet; all data is currently static or client-side.

---

## Tech Stack

- **Frontend:** Next.js
- **UI & Charts:** Shadcn
- **PDF Generation:** jsPDF (or any suitable library)

---

## Usage

1. Open the website and access the dashboard.
2. Enter your daily or monthly sales data.
3. Generate PDF invoices for any transaction.
4. View profits in charts, divided by quarters.
